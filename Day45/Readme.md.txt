Solved an interesting array manipulation problem from CodeChef where I had to merge two lists in a custom order:
List1 + List2 + List1

📥 Example Input:
5 3  
1 3 5 3 9  
3 2 1
📤 Output:
1 3 5 3 9 3 2 1 1 3 5 3 94

💡 Approach:
1. Read N and X using Scanner
2.Store List 1 and List 2
3.Merge as: List1 → List2 → List1 again
4.Print the final merged list

🛠️ Concepts Used:
1.Arrays
2.Index manipulation
3.Input/Output handling

⏳ Time Complexity:O(N + X)
📦 Space Complexity:O(N + X + N)
