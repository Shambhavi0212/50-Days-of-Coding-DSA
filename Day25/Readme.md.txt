🧩 Problem:  
Given a number N and a digit count D, find the largest number less than N formed using exactly D digits from N. The digits must come from N, rearranged without repeating positions. No leading zeros. Output must be strictly less than N.

🧪 Examples:  
Input: 1235  
3  
Output: 1229

Input: 1235  
2  
Output: 1199

Input: 1235  
1  
Output: 999

⏱ Time Complexity: O(N!)  
📦 Space Complexity: O(N)
