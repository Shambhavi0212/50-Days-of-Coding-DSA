🧩 Problem Solved: Insert Element at First and Last
📊 Problem Type: Insertion | Array | Input/Output | Basic Manipulation 

📝 Today, I:
Solved a basic but insightful array manipulation problem from CodeChef, where I had to insert a given number at both the start and end of an array.

📥 Example Input:
1 3 5 7 9 
4
📤 Output: 4 1 3 5 7 9 4
💡 Approach:
1. Read input using Scanner
2.Create a new array of size N + 2
3.Insert X at the beginning and end
4.Copy original array elements in the middle
5.Print the result

🛠️ Concepts Used:
1.Arrays
2.Input/Output handling
3.Indexing logic

⏳ Time Complexity:O(N)
📦 Space Complexity:O(N + 2)
