🧩 Problem Solved:Toggle Only Vowels in a String (SPEC)

🧑‍🏫 Source:CRT Practice Question

🧩 Problem:
Given a sentence S,toggle the case of only the vowels (a,e,i,o,u — both uppercase and lowercase).

Convert lowercase vowels to uppercase.

Convert uppercase vowels to lowercase.

All consonants and spaces remain unchanged.

🧪 Examples:
Input: hello EverYone
Output: hEllO evErYOnE

Input: aaaBBBBiiii
Output: AAABBBBIIII

⏱ Time Complexity:O(n)
📦 Space Complexity:O(1)
