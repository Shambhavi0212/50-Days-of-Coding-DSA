🧩 Problem Solved : Longest Word in a Sentence

🔗 Hackerrank Submission Link : https://www.hackerrank.com/contests/srings-level-1-spec/challenges/longest-word-in-a-sentencespec/submissions/code/1392343048

🧩 Problem:
Given a sentence, return the word with the maximum length.
If multiple words have the same maximum length, return the first one among them.

🧪 Examples:
Input: "Hai iam Adam from Hyderabad" → Output: ✅ Hyderabad
Input: "hope all are doing well" → Output: ✅ doing

⏱ Time Complexity:O(n) – where n is the number of characters in the string
📦 Space Complexity:O(n) – for storing split words in an array
