🧩 Problem Solved : Reverse Words Starting with Consonants

🧑‍🏫 Source:HackerRank (Given by Sarfaraz Sir)

🧩 Problem:
Given a sentence S, reverse only the words that start with a consonant.Words starting with vowels (a, e, i, o, u - case insensitive) remain unchanged.
If no word starts with a consonant, print:
No Vowel Starting Word is present in String

🧪 Examples:
Input:  Hello Everyone how are Doing
Output: olleH Everyone woh are gnioD

Input:  kumar went to Hyd
Output: No Vowel Starting Word is present in String

⏱ Time Complexity: O(n*m)
    where:
  n = number of words in the sentence..
  m = average length of each word...
📦 Space Complexity: O(1)
